# Page Flip Animation Frames

Place your page flip animation frames here with the following naming:

- `page_flip_01.png`
- `page_flip_02.png`
- `page_flip_03.png`
- `page_flip_04.png`
- `page_flip_05.png`
- `page_flip_06.png`
- `page_flip_07.png`
- `page_flip_08.png`
- `page_flip_09.png`
- `page_flip_10.png`

## Specifications
- **Format:** PNG with alpha channel
- **Size:** 400x480 pixels (recommended)
- **Animation:** Page flipping from left to right
- **Style:** Old paper texture with slight curl

The system will automatically detect and load these frames for the animation.
If frames are not present, a fallback tween animation will be used instead.
